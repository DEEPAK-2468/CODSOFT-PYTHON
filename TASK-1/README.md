# Task 1 - To-Do List Application

## Description

This project is a command-line based To-Do List Application developed using Python. It helps users manage their daily tasks efficiently by allowing them to add, view, update, and delete tasks.

## Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Delete tasks
* Automatically save tasks to a file
* Load previously saved tasks when the application starts

## Technologies Used

* Python
* File Handling
* Dictionary Data Structure

## Files

* task_manager.py - Main source code
* Task.txt - Stores task data

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Run the Python file:

```bash
python task_manager.py
```

## Sample Menu

1. Add Task
2. View Task
3. Mark Task as Complete
4. Delete Task
5. Exit
